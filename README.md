## Features Implemented:

- Login/Logout
- Register
- Add Issue
- Edit Issue
- See all issues
- Add and see comments for each issue
- Upvote feature
- Add multiple images for a issue
- Update status for each issue as an admin

## Features Pending:

- Email Verification
- Deletion of issues as an admin
